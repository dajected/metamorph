Focused by design

A progression engine, not a story director.

Your character model still decides what happens. Metamorph keeps the progression consistent and gives World Info the right key at the right time.

01

Monotonic progress

Stats only move forward. Each affected stat can increase once per assistant message, and resetting is the only way back.

02

New-change detection

The helper judge scans the latest response and uses compact memory to reject repeated or already-counted changes.

03

One active tier

The highest satisfied tier wins. Only its exact key enters context, keeping World Info activation predictable.

Installation

Drop it into SillyTavern.

Metamorph requires SillyTavern 1.18.0 or later and an existing API connection when helper judging is enabled.

  1. Download the latest main ZIP and extract it.
  2. Copy the metamorph folder into your third-party extensions directory.public/scripts/extensions/third-party/metamorph/
  3. Reload SillyTavern and enable Metamorph under Manage Extensions.
  4. Set up tier keys in World Info, then import or build your tracker setup.
